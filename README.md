# MessageChannel-Web

### 这个项目是 [Messagechannel](https://github.com/LucasZhanye/MessageChannel)的前端页面，使用了以下技术栈：

- vue3
- element-plus
- vue-router
- pinia
- axios
- typescript

### 主要包含了用户登录，客户端列表，订阅列表等功能，功能比较简单，以下是页面的部分图片
